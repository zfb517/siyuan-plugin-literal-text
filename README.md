# 转义 / Escape

![Version](https://img.shields.io/badge/version-2.8.2-brightgreen)

![Min App](https://img.shields.io/badge/siyuan-%E2%89%A5%203.7.0-blue)

思源笔记插件 — 输入什么就显示什么，不让 Markdown 自动渲染吃掉你的 `*` `#` 等符号。

---

## 功能

### 1. 字面文本输入

输入一段文本，原样插入，不被 Markdown 渲染。点击 `Ctrl+Shift+L`（或斜杠命令）会弹出输入窗，提供两种模式：

- **行内代码**（默认）：整体包裹为 `` `代码` ``，浅灰底框，适合含 `*` `#` 的型号/规格。
- **转义字符**：对文本做 Markdown 转义后插入，所见即所得地显示为纯文本，例如：

| 输入      | 实际插入         | 效果         |
| ------- | ------------ | ---------- |
| `*`     | `\*`         | 不被渲染为斜体/粗体 |
| `#`     | `` `#` ``    | 不被渲染为标签/标题 |
| `a*b#c` | ``a`#`b\*c`` | 混合符号均被保护   |

- **快捷键**：`Ctrl+Shift+L`
- **斜杠命令**：`/字面` `/literal`（行内代码）、`/转义` `/escape`（转义字符）

### 2. 自动转义（默认开启）

开启后，键盘输入指定字符时自动转义，阻止 Markdown 渲染（字符集可在设置中勾选，默认 `*` 和 `#`；**代码块内输入的字符不受影响**）：

| 输入  | 实际插入      | 效果         |
| --- | --------- | ---------- |
| `*` | `\*`      | 不被渲染为斜体/粗体 |
| `#` | `` `#` `` | 不被渲染为标签/标题 |

- **快捷键**：`Ctrl+Shift+E` 切换开关
- **状态指示**：顶部工具栏第三个按钮（绿色盾牌=ON / 灰色空心=OFF）

> **提示**：`#` 字符在思源 Lute 引擎中用 `\#` 或零宽空格均无法阻止解析，因此采用行内代码包裹 `` `#` ``，视觉上会有浅灰背景，但这是唯一可靠方案。

> **兼容性提示**：自动转义监听已对输入法合成（中文/日文等）做跳过处理，使用输入法打字时不会被误拦截。

### 3. 富文本粘贴

粘贴公众号/网页内容时自动下载图片到思源笔记。

- **快捷键**：`Ctrl+Shift+V`
- **斜杠命令**：`/富文本` 或 `/richpaste`

> 移动端 `Ctrl+Shift+V` 不可用，请直接 `Ctrl+V` 粘贴，插件会通过事件拦截自动处理富文本内容。

- **图片保存子目录**：在设置中填写子目录名（如 `wechat`），图片会存到 `assets/wechat/` 而非默认 `assets/`，便于按来源归类（留空则用默认 `assets/`）。

### 4. 选区转字面量（L1）

选中一段已存在的文本，一键将其转换为「不被渲染」的字面量，无需弹窗：

- **行内代码**：选中文本整体包裹为 `` `代码` ``（浅灰底框）。
- **转义字符**：选中文本做 Markdown 转义后原地替换为纯文本（所见即所得）。
- **入口**：
  - 顶栏第 4 个按钮「选区转转义」（纯文本模式）
  - 斜杠命令：`/选区转字面`（行内代码）、`/选区转转义`（纯文本）
  - 命令面板：`选区转字面量（行内代码）`、`选区转转义（纯文本）`

> 适合把文档里已经写好的型号/规格（如 `JTW-ZD-9911*2`）就地变成字面量。

### 5. 字符全半角转换（L2）

选中文本，一键在**全角 ⇄ 半角**之间转换，解决从 PDF / Word 粘贴时数字字母变成全角（如 `１．５`）的问题：

- **全角转半角**：`１．５ｋＷ` → `1.5kW`
- **半角转全角**：`1.5kW` → `１．５ｋＷ`
- **入口**：
  - 斜杠命令：`/全角转半角`、`/半角转全角`
  - 命令面板：`全角转半角`、`半角转全角`

> 覆盖 ASCII 可见字符（！～）与全角空格，中文及非常规字符原样保留。

### 6. 字面文本块（多行）

选中「字面文本块」斜杠命令（或命令面板），弹窗输入多行文本，插入一个**代码块**节点，内容原样显示、不被 Markdown 渲染：

- 适合消防设备型号表、多行规格、长代码片段等需要在块内保留 `*` `#` 等符号的场景
- **入口**：斜杠命令 `/字面块` `/literalblock`、命令面板「字面文本块（多行）」

### 7. 反字面（还原为普通文本）

把已转成字面量的文本还原：选中文本后执行「反字面」，会同时去除行内代码包裹与转义反斜杠，变回普通文本，与「选区转字面量」形成闭环：

- **入口**：斜杠命令 `/反字面` `/unwrap`、命令面板「反字面（还原为普通文本）」

## 安装

### 方式一：从 GitHub Release 安装

1. 下载 [siyuan-plugin-literal-text.zip](https://github.com/zfb517/siyuan-plugin-literal-text/releases)
2. 思源笔记 → 设置 → 插件 → 从本地安装 → 选择 zip 文件

### 方式二：从源码构建

```bash
git clone https://github.com/zfb517/siyuan-plugin-literal-text.git
cd siyuan-plugin-literal-text
node esbuild-build.js
```

然后将以下文件打包为 zip 安装：

```
index.js  index.css  plugin.json  icon.png  i18n/en.json  i18n/zh-CN.json
```

## 开发

```bash
# 安装依赖
npm install

# 一次性构建
node esbuild-build.js

# 监听模式（修改代码自动构建）
node esbuild-build.js --watch
```

项目结构：

```
src/index.js        # 插件源码
index.css           # 样式
plugin.json         # 插件元信息
icon.png            # 160x160 插件图标
i18n/
  zh-CN.json        # 中文翻译
  en.json           # 英文翻译
esbuild-build.js    # 构建脚本
```

## 开发

- 构建（esbuild 输出 CJS 到根目录 `index.js`）：`npm run build`
- 类型检查（严格模式，零错误交付）：`npm run check`（`tsc --noEmit`）
- 代码本体为 `src/index.ts`，`src/globals.d.ts` 声明了思源运行时注入的全局 `$`（jQuery）类型
