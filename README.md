# 字面文本+富粘贴 (Literal Text + Rich Paste)

思源笔记插件：快速输入 `*#` 等特殊符号不被 Markdown 渲染 + 粘贴公众号/网页文章自动下载图片。

## 功能

### 1. 字面文本输入
- `Ctrl+Shift+L` 弹窗输入，内容以行内代码或转义字符插入，不被 Markdown 渲染
- `Ctrl+Shift+E` 切换自动转义模式（输入 `*` `#` 自动加 `\`）
- 斜杠命令：`/字面` `/转义` `/literal` `/escape`

### 2. 富文本粘贴
- 自动拦截粘贴，调用思源内核 API 将 HTML 转 Markdown 并下载图片到本地
- 支持公众号文章、网页内容
- `Ctrl+Shift+V` 手动触发
- 斜杠命令：`/富文本` `/richpaste`

### 3. 设置
- 斜杠命令：`/设置` `/setting`

## 安装

1. 解压 zip 文件
2. 思源笔记 → 设置 → 集市 → 已安装 → 从文件系统安装
3. 选择解压后的文件夹（含 plugin.json 的目录）
4. 启用插件

## 平台支持

| 平台 | 支持 | 说明 |
|------|------|------|
| Windows/Linux/macOS | ✅ | 桌面端完整功能 |
| 浏览器桌面端 | ✅ | 完整功能 |
| 移动端（含鸿蒙 NEXT） | ✅ | 斜杠命令+粘贴拦截；快捷键需外接键盘 |

## 技术说明

- 粘贴处理使用思源官方事件总线 `eventBus.on("paste")`，桌面端和移动端通用
- 图片下载由思源内核执行（`/api/extension/copy`），不受浏览器 CORS 限制
- 若公众号图片丢失，可能是微信 CDN 防盗链导致内核下载失败
