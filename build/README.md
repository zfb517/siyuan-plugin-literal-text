# 字面文本 + 富粘贴 插件

思源笔记插件，解决两个痛点：

1. **字面文本输入**：快速输入 `*#JTW-ZD-9911` 等特殊符号，不被 Markdown 渲染
2. **富文本粘贴**：粘贴公众号/网页内容时，自动下载图片到本地 assets 目录

## 功能一：字面文本输入

| 操作 | 说明 |
|------|------|
| `Ctrl+Shift+L` | 弹窗输入，结果以行内代码或转义字符插入 |
| `Ctrl+Shift+E` | 切换自动转义模式（开启后输入 `*` `#` 自动加 `\`） |
| 选中文本后按 `Ctrl+Shift+L` | 直接包裹为行内代码 |
| 斜杠命令 `/字面` `/literal` | 弹窗输入 |
| 顶部栏图标 | 点击打开弹窗 |

**两种插入模式**（弹窗中可切换）：
- **行内代码**：用反引号包裹，显示灰色底框样式，推荐
- **转义字符**：在特殊符号前加 `\`，纯文本显示

## 功能二：富文本粘贴

粘贴公众号文章或网页时，自动调用思源内核 API（`/api/extension/copy`），将 HTML 转为 Markdown **并自动下载其中的图片到本地 assets 目录**。

| 操作 | 说明 |
|------|------|
| 直接 `Ctrl+V` 粘贴 | 自动拦截，调用内核处理（需在设置中启用） |
| `Ctrl+Shift+V` | 手动触发富粘贴（备用） |
| 斜杠命令 `/富文本` `/richpaste` | 手动触发 |
| 顶部栏图标（图片图标）| 手动触发 |

**两种粘贴模式**（设置中切换）：
- **Markdown 模式**（推荐）：HTML 转 Markdown，图片本地化，文本可编辑
- **HTML 块模式**：原样保留所有视觉格式，插入为 HTML 块

## 安装方法

1. 下载 `siyuan-plugin-literal-text.zip`
2. 打开思源笔记 → 设置 → 集市 → 已安装 → 点击「从文件系统安装」
3. 选择下载的 `.zip` 文件
4. 安装完成后启用插件

或手动安装：
1. 解压 `.zip` 到思源插件目录（`工作空间/data/plugins/`）
2. 重启思源

## 注意事项

- 富粘贴功能需要思源内核能访问外网（下载图片）
- 微信公众号图片 CDN（mmbiz.qpic.cn）有防盗链，若自动下载失败，插件会提示并建议手动处理
- 若粘贴无反应，请检查设置中是否启用了「自动富文本粘贴」
